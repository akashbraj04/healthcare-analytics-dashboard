# Healthcare Analytics Dashboard
## Project Overview

This project analyzes healthcare operational and patient datasets to provide actionable insights through SQL, Python, and Power BI.

The dashboard helps monitor key healthcare KPIs, identify performance trends, improve reporting visibility, and support data-driven decision making.
## Business Problem

Healthcare organizations generate large volumes of operational and patient-related data.

The objective was to transform raw healthcare data into meaningful KPIs and interactive dashboards to support operational monitoring and performance improvement.
## Tools & Technologies

- Python
- Pandas
- NumPy
- Power BI
- DAX
- Power Query
- Excel
## Key Features

- KPI Dashboard Development
- Patient & Operational Analytics
- Trend Analysis
- Data Cleaning & Validation
- Interactive Filtering
- Power BI Visualizations
- Automated Reporting Workflows
## Business Impact

- Processed and analyzed 500K+ healthcare records
- Improved KPI reporting visibility
- Reduced manual reporting effort through automation
- Enabled faster operational decision making
- Improved data quality and reporting consistency
